Meet Delta.


An open-source music bot designed for you, not for money.

Secure.


Delta is open source, meaning that you can view the code and see how we're using your data, and if you like, make your own version of the bot.

Easy to use.


Delta provides a dashboard to control your music with ease, no more remembering complicated syntax for commands.

Upgradeable.


Using our simple guide, learn how you can host part of the bot to help contribute to everyone's experience..

Free forever.


We never plan to introduce a premium tier to Delta, so you can listen unrestricted.
